### Descrição geral das funcionalidades
- Criar conta
	-	Registra um novo usuário
- Cadastrar
	-	Adiciona uma nova dívida
- Editar
	-	Altera dívida existente
- Remover
	-	Remove uma dívida
- Pagar
	- Informa que a dívida foi paga e atualiza a data de vencimento
